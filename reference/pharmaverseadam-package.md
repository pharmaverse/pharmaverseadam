# pharmaverseadam: ADaM Test Data for the 'Pharmaverse' Family of Packages

A set of Analysis Data Model (ADaM) datasets constructed using the Study
Data Tabulation Model (SDTM) datasets contained in the 'pharmaversesdtm'
package and the template scripts from the 'admiral' family of packages.
ADaM dataset specifications are described in the CDISC ADaM
implementation guide, accessible by creating a free account on
<https://www.cdisc.org/>.
